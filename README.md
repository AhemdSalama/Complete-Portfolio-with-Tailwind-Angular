# 🚀 Modern Developer Portfolio | Ahmed Salama

Welcome to my personal portfolio repository! This project is a fully responsive, highly interactive web developer and AI automation engineer portfolio built with **Angular** and **Tailwind CSS**.

## ✨ Key Features

* **🎨 Modern UI/UX:** Built with Glassmorphism concepts, smooth gradients, and elegant hover animations.
* **🌗 Custom Theme Switcher:** A dynamic modal allowing users to switch between multiple color themes (Ocean Blue, Sunset Orange, Forest Green, etc.).
* **📂 Dynamic Project Filtering:** A seamless, animated grid to filter projects by categories (Web, AI, Automation).
* **📄 Single Project Details Page:** Dynamic routing that passes parameters to a dedicated "Case Study" style page for each project.
* **📱 Fully Responsive:** Carefully crafted with Tailwind CSS to look stunning on mobile, tablet, and desktop screens.
* **🚀 Anchor Scrolling:** Smooth internal navigation across sections (Hero, About, Projects, Contact).

## 🛠️ Tech Stack

* **Framework:** Angular (Standalone Components)
* **Styling:** Tailwind CSS
* **Language:** TypeScript, HTML5
* **Routing:** Angular Router (with `withInMemoryScrolling` & `withAnchorScrolling`)
